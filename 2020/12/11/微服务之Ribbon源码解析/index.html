дәӢжғ…зҡ„иө·еӣ жҳҜиҝҷж ·зҡ„пјҢе…¬еҸёеҶ…йғЁиҰҒе®һзҺ°еҹәдәҺZuulзҪ‘е…ізҡ„зҒ°еәҰи·Ҝз”ұпјҢеңЁдёҠзәҝж—¶иҝӣиЎҢзҒ°еәҰжөӢиҜ•пјҢж•…йңҖиҰҒй…ҚзҪ®дёҡеҠЎеҫ®жңҚеҠЎеҗ‘EurekaжіЁеҶҢзҡ„metadataе…ғж•°жҚ®пјҢе’ҢиҮӘе®ҡд№үRibbonзҡ„иҙҹиҪҪи§„еҲҷиҫҫеҲ°еҸӘи®ҝй—®зҒ°еәҰжңҚеҠЎзҡ„зӣ®зҡ„гҖӮиҝҷж ·е°ұйңҖиҰҒиҮӘе®ҡд№үRibbonзҡ„IRuleпјҢе®һзҺ°зҒ°еәҰиҜ·жұӮеҸӘдјҡиҙҹиҪҪеҲ°еёҰжңүзҒ°еәҰж Үзӯҫе…ғж•°жҚ®зҡ„дёҡеҠЎеҫ®жңҚеҠЎдёҠпјҢеҪ“иҮӘе®ҡд№үIRuleи§„еҲҷејҖеҸ‘еҘҪеҗҺпјҢй—®йўҳжҳҜеҰӮдҪ•е°ҶиҝҷдёӘIRuleи§„еҲҷй…ҚзҪ®з»ҷжҹҗдёӘRibbon ClientжҲ–иҖ…е…ЁеұҖз”ҹж•ҲгҖӮ

жң¬ж¬ЎдҪҝз”ЁSpring Cloud Dalston.SR5зүҲжң¬

еңЁе…¶ е®ҳж–№ж–ҮжЎЈ дёӯе…¶е®һе·Із»Ҹз»ҷеҮәдәҶдёҖдәӣеҰӮдҪ•й’ҲеҜ№жҹҗдёӘClient жҲ–иҖ… дҝ®ж”№й»ҳи®Өй…ҚзҪ®зҡ„ж–№ејҸпјҢдҪҶжІЎжңүиҜҙжҳҺдёәд»Җд№Ҳиҝҷж ·дҪҝз”Ё

  • з®ҖеҚ•еҲҶжһҗSpring Cloud RibbonеҗҜеҠЁж—¶еҰӮдҪ•иҮӘеҠЁй…ҚзҪ®зҡ„пјҢд»ҘдәҶи§Је…¶иЈ…й…ҚеҲ°Springдёӯзҡ„Bean
  • Spring Cloud Ribbon Clientзҡ„жҮ’еҠ иҪҪ
  • Spring Cloud Ribbon Clientзҡ„й…ҚзҪ®еҠ иҪҪпјҢеҢ…еҗ«е…ЁеұҖй…ҚзҪ®еҸҠClientй…ҚзҪ®
  • еҰӮдҪ•иҮӘе®ҡд№үClientй…ҚзҪ®гҖҒе…ЁеұҖй…ҚзҪ®

    Spring Cloud RibbonиҮӘеҠЁй…ҚзҪ®

еҪ“еүҚзүҲжң¬дёӯзҡ„NetflixжүҖжңүиҮӘеҠЁй…ҚзҪ®йғҪеңЁspring-cloud-netflix-core-xxx.jarдёӯпјҢж №жҚ®е…¶META-INF/spring.factoriesдёӯзҡ„й…ҚзҪ®еҫ—зҹҘпјҢSpring Cloud Ribbonзҡ„иҮӘеҠЁй…ҚзҪ®зұ»дёә RibbonAutoConfiguration

RibbonAutoConfiguration

дёҠйқўRibbonAutoConfigurationеҲӣе»әзҡ„Beanдё»иҰҒеҲҶд»ҘдёӢеҮ зұ»пјҡ

  • дёәRibbon ClientеҲӣе»әзҺҜеўғеҸҠиҺ·еҸ–й…ҚзҪ®
    • SpringClientFactoryпјҡ дјҡз»ҷжҜҸдёӘRibbon ClientеҲӣе»әдёҖдёӘзӢ¬з«Ӣзҡ„Springеә”з”ЁдёҠдёӢж–ҮApplicationContextпјҢе№¶еңЁе…¶дёӯеҠ иҪҪеҜ№еә”зҡ„й…ҚзҪ®еҸҠRibbonж ёеҝғжҺҘеҸЈзҡ„е®һзҺ°зұ»
    • PropertiesFactoryпјҡ з”ЁдәҺд»ҺSpring enviromentзҺҜеўғдёӯиҺ·еҸ–й’ҲеҜ№жҹҗдёӘRibbon Clientй…ҚзҪ®зҡ„ж ёеҝғжҺҘеҸЈе®һзҺ°зұ»пјҢе№¶е®һдҫӢеҢ–
  • еҲӣе»әRibbonLoadBalancerClientпјҢе№¶е°ҶspringClientFactoryжіЁе…ҘпјҢж–№дҫҝд»ҺдёӯиҺ·еҸ–еҜ№еә”зҡ„й…ҚзҪ®еҸҠе®һзҺ°зұ»пјҢRibbonLoadBalancerClientжҳҜSpringеҜ№LoadBalancerClientжҺҘеҸЈзҡ„е®һзҺ°зұ»пјҢе…¶execute()ж–№жі•жҸҗдҫӣе®ўжҲ·з«ҜиҙҹиҪҪеқҮиЎЎиғҪеҠӣ
  • жҮ’еҠ иҪҪзӣёе…і
    • RibbonEagerLoadPropertiesпјҡ жҮ’еҠ иҪҪй…ҚзҪ®йЎ№PropertiesпјҢеҸҜд»ҘжҢҮе®ҡжҳҜеҗҰжҮ’еҠ иҪҪпјҢеҸҠе“ӘдәӣClientдёҚжҮ’еҠ иҪҪ
    • RibbonApplicationContextInitializerпјҡ еҗҜеҠЁж—¶е°ұеҠ иҪҪRibbonClientй…ҚзҪ®пјҲйқһжҮ’еҠ иҪҪпјүзҡ„еҲқе§ӢеҢ–еҷЁ

еҸҜд»ҘзңӢеҲ°й»ҳи®ӨеҗҜеҠЁжөҒзЁӢдёӯе№¶жІЎжңүеҠ иҪҪRibbonClientзҡ„дёҠдёӢж–Үе’Ңй…ҚзҪ®дҝЎжҒҜпјҢиҖҢжҳҜеңЁдҪҝз”Ёж—¶жүҚеҠ иҪҪпјҢеҚіжҮ’еҠ иҪҪ

Spring Cloud RibbonClientзҡ„жҮ’еҠ иҪҪ

ж—ўз„¶жҳҜеңЁдҪҝз”Ёж—¶жүҚдјҡеҠ иҪҪпјҢйӮЈд№Ҳд»ҘZuulзҪ‘е…ідёәдҫӢпјҢеңЁе…¶RibbonRoutingFilterдёӯдјҡеҲӣе»әRibbonCommandпјҢе…¶еҢ…еҗ«дәҶRibbonзҡ„иҙҹиҪҪеқҮиЎЎ

еңЁжү§иЎҢRibbonRoutingFilter#run()иҝӣиЎҢи·Ҝз”ұж—¶дјҡжү§иЎҢforward()ж–№жі•пјҢз”ұдәҺжӯӨеӨ„жҳҜеңЁHystrixCommandеҶ…йғЁжү§иЎҢRibbonиҙҹиҪҪеқҮиЎЎи°ғз”ЁпјҢж•…дҪҝз”ЁribbonCommandFactoryеҲӣе»әRibbonCommandпјҢRibbonе®ўжҲ·з«Ҝзҡ„жҮ’еҠ иҪҪе°ұеңЁиҝҷдёӘж–№жі•еҶ…пјҢиҝҷйҮҢжҲ‘д»¬зңӢHttpClientRibbonCommandFactoryе®һзҺ°зұ»

еҲӣе»әRibbonLoadBalancingHttpClientзҡ„йҖ»иҫ‘еңЁ SpringClientFactory#getClient(serviceId, RibbonLoadBalancingHttpClient.class)пјҢеҰӮдёӢпјҡ

  • SpringClientFactory#getInstance(name, clientClass)
    • NamedContextFactory#getInstance(name, type)пјҡ
      • иҺ·еҸ–ClientеҜ№еә”зҡ„ApplicationContextпјҢеҰӮжІЎжңүеҲҷи°ғз”ЁcreateContext()еҲӣе»әпјҢе…¶дёӯеҢ…еҗ«жіЁеҶҢз»ҹдёҖй»ҳи®Өй…ҚзҪ®зұ»RibbonClientConfigurationпјҢжҲ–@RibbonClientгҖҒ@RibbonClients(defaultConfiguration=xxx) и®ҫзҪ®зҡ„й…ҚзҪ®зұ»зҡ„йҖ»иҫ‘
      • д»ҺApplicationContextдёӯж №жҚ®зұ»еһӢиҺ·еҸ–е®һдҫӢпјҢеҰӮжІЎжңүдҪҝз”ЁеҸҚе°„еҲӣе»әпјҢе№¶йҖҡиҝҮIClientConfigй…ҚзҪ®

еҰӮдёҠжү§иЎҢе®ҢжҜ•RibbonClientе°ұеҹәжң¬жҮ’еҠ иҪҪе®ҢжҲҗдәҶпјҢе°ұеҸҜд»ҘеҲ°RibbonClientеҜ№еә”зҡ„ApplicationContextдёӯз»§з»ӯиҺ·еҸ–е…¶е®ғж ёеҝғжҺҘеҸЈзҡ„е®һзҺ°зұ»дәҶпјҢиҝҷдәӣе®һзҺ°зұ»йғҪжҳҜж №жҚ® й»ҳи®Ө/е…ЁеұҖ/ClientиҮӘе®ҡд№ү й…ҚзҪ®еҲӣе»әзҡ„

дёҠйқўжҜ”иҫғйҮҚиҰҒзҡ„е°ұжҳҜеңЁеҲӣе»әжҜҸдёӘRibbonClientзҡ„ApplicationContextзҡ„createContext(name)ж–№жі•пјҢе…¶дёӯеҢ…еҗ«дәҶж №жҚ®е“ӘдёӘ@Configurationй…ҚзҪ®зұ»еҲӣе»әRibbonж ёеҝғжҺҘеҸЈзҡ„е®һзҺ°зұ»зҡ„йҖ»иҫ‘пјҢж•…йңҖйҮҚзӮ№еҲҶжһҗпјҲRibbonж ёеҝғжҺҘеҸЈи®Іи§Ј еҸӮиҖғпјү

йӮЈд№ҲеңЁcreateContext(name)ж–№жі•еҲӣе»әеҪ“еүҚRibbon Clientзӣёе…ізҡ„дёҠдёӢж–ҮпјҢе№¶жіЁе…Ҙй…ҚзҪ®зұ»ж—¶пјҢйҷӨдәҶй»ҳи®Өй…ҚзҪ®зұ»RibbonClientConfigurationжҳҜеҶҷжӯ»зҡ„пјҢе…¶е®ғзҡ„й…ҚзҪ®зұ»пјҢеҰӮdefaultе…ЁеұҖй…ҚзҪ®зұ»пјҢй’ҲеҜ№жҹҗдёӘRibbon Clientзҡ„й…ҚзҪ®зұ»пјҢеҸҲжҳҜжҖҺд№Ҳй…ҚзҪ®зҡ„е‘ўпјҹ

Spring Cloud RibbonClientзҡ„й…ҚзҪ®еҠ иҪҪпјҢеҢ…еҗ«е…ЁеұҖй…ҚзҪ®еҸҠClientй…ҚзҪ®

еҲӣе»әRibbonClientеҜ№еә”ApplicationContextпјҢе№¶жіЁеҶҢжүҖжңүеҸҜз”Ёзҡ„Configurationй…ҚзҪ®зұ»

ж №жҚ®еҰӮдёҠйҖ»иҫ‘еҸҜд»ҘзңӢеҮәдјҡд»Һ3дёӘең°ж–№е°ҶRibbonзӣёе…ізҡ„Configurationй…ҚзҪ®зұ»жіЁеҶҢеҲ°дё“й—Ёдёәе…¶еҮҶеӨҮзҡ„ApplicationContextдёҠдёӢж–ҮпјҢе№¶ж №жҚ®й…ҚзҪ®зұ»еҲӣе»әRibbonж ёеҝғжҺҘеҸЈзҡ„е®һзҺ°зұ»пјҢеҚіиҫҫеҲ°й…ҚзҪ®RibbonClientзҡ„зӣ®зҡ„

  1. д»ҺconfigurationsиҝҷдёӘMapдёӯж №жҚ®RibbonClient nameиҺ·еҸ–дё“й—Ёдёәе…¶жҢҮе®ҡзҡ„configurationй…ҚзҪ®зұ»пјҢе№¶жіЁеҶҢеҲ°е…¶еҜ№еә”зҡ„ApplicationContextдёҠдёӢж–Ү
  2. д»ҺconfigurationsиҝҷдёӘMapдёӯжүҫеҲ° default. ејҖеӨҙ зҡ„й…ҚзҪ®зұ»пјҢеҚідёәжүҖжңүRibbonClientзҡ„й»ҳи®Өй…ҚзҪ®пјҢе№¶жіЁеҶҢеҲ°е…¶еҜ№еә”зҡ„ApplicationContextдёҠдёӢж–Ү
  3. еҰӮжһңдёҚжҳҜејҖеҸ‘иҖ…еҚ•зӢ¬жҢҮе®ҡзҡ„иҜқпјҢеүҚдёӨйЎ№йғҪжҳҜжІЎжңүж•°жҚ®зҡ„пјҢиҝҳдјҡжіЁеҶҢSpring Cloudзҡ„й»ҳи®Өй…ҚзҪ®зұ»RibbonClientConfiguration

йӮЈд№ҲconfigurationsиҝҷдёӘMapйҮҢзҡ„й…ҚзҪ®зұ»ж•°жҚ®жҳҜд»Һе“Әе„ҝжқҘзҡ„е‘ўпјҹпјҹдёӢйқўйҖҗжӯҘеҲҶжһҗ

йҰ–е…ҲжҳҜеңЁRibbonAutoConfigurationиҮӘеҠЁй…ҚзҪ®зұ»еҲӣе»әSpringClientFactoryжҳҜи®ҫзҪ®зҡ„пјҢиҝҷдёӘconfigurationsйӣҶеҗҲжҳҜ@Autowiredзҡ„Springе®№еҷЁеҶ…зҡ„RibbonClientSpecificationйӣҶеҗҲпјҢйӮЈд№ҲRibbonClientSpecificationйӣҶеҗҲжҳҜдҪ•ж—¶иў«жіЁеҶҢзҡ„пјҹпјҹ

еҰӮдёҠеҸҜзҹҘпјҢconfigurationsй…ҚзҪ®зұ»йӣҶеҗҲжҳҜж №жҚ®@RibbonClient е’Ң @RibbonClients жіЁи§Јй…ҚзҪ®зҡ„пјҢеҲҶеҲ«жңү й’ҲеҜ№е…·дҪ“жҹҗдёӘRibbonClientзҡ„й…ҚзҪ® е’Ң defaultй»ҳи®Өй…ҚзҪ®

жҖ»з»“дёҖдёӢпјҢRibbonзӣёе…ізҡ„@Configurationй…ҚзҪ®зұ»жҳҜеҰӮдҪ•еҠ иҪҪзҡ„

  1. еңЁеҲӣе»әе®ҢRibbonClientеҜ№еә”зҡ„AnnotationConfigApplicationContextеҗҺпјҢе…Ҳд»Һж №жҚ®@RibbonClient е’Ң @RibbonClients жіЁи§ЈеҠ иҪҪзҡ„configurationsйӣҶеҗҲдёӯжүҫеҪ“еүҚRibbonClient nameеҜ№еә”зҡ„й…ҚзҪ®зұ»пјҢеҰӮжңүпјҢе°ұжіЁеҶҢеҲ°дёҠдёӢж–Ү
  2. еҶҚд»ҺconfigurationsйӣҶеҗҲдёӯжүҫж №жҚ®@RibbonClientsжіЁи§ЈеҠ иҪҪзҡ„ default.ејҖеӨҙ зҡ„й»ҳи®Өй…ҚзҪ®зұ»пјҢеҰӮжңүпјҢе°ұжіЁеҶҢеҲ°дёҠдёӢж–Ү
  3. жңҖеҗҺжіЁеҶҢSpring Cloudй»ҳи®Өзҡ„ RibbonClientConfiguration

дёҠйқўиҜҙжҳҜеҰӮдҪ•еҲӣе»әRibbonClientзӣёе…ізҡ„ApplicationContextдёҠдёӢж–ҮеҸҠжіЁеҶҢRibbon Clientзӣёе…ізҡ„й…ҚзҪ®зұ»зҡ„йҖ»иҫ‘пјҢеңЁзЎ®е®ҡй…ҚзҪ®зұ»еҗҺпјҢе…¶дёӯдјҡз”ЁеҲ°Ribbonзҡ„IClientConfigзӣёе…ізҡ„е®ўжҲ·з«Ҝй…ҚзҪ®жқҘеҠ иҪҪRibbonе®ўжҲ·з«Ҝзӣёе…ізҡ„й…ҚзҪ®дҝЎжҒҜпјҢеҰӮи¶…ж—¶й…ҚзҪ®гҖҒе…·дҪ“еҲӣе»әе“ӘдёӘж ёеҝғжҺҘеҸЈзҡ„е®һзҺ°зұ»зӯүпјҢеҸҜд»Ҙд»ҺSpring Cloudй»ҳи®ӨжіЁеҶҢзҡ„ RibbonClientConfigurationжқҘдёҖжҺўз©¶з«ҹ

RibbonClientConfigurationй…ҚзҪ®еҠ иҪҪеҸҠRibbonж ёеҝғжҺҘеҸЈе®һзҺ°зұ»еҲӣе»ә

дёҠйқўеҸӘжҲӘеҸ–дәҶдёҖж®өд»Јз ҒпјҢз»ҷеҮәдәҶRibbonзӣёе…ізҡ„ IClientConfigе®ўжҲ·з«Ҝй…ҚзҪ® е’Ң жҹҗдёҖдёӘж ёеҝғжҺҘеҸЈIRuleе®һзҺ°зұ» жҳҜеҰӮдҪ•еҠ иҪҪй…ҚзҪ®е№¶еҲӣе»әзҡ„

IClientConfig

IClientConfigе°ұжҳҜRibbonе®ўжҲ·з«Ҝй…ҚзҪ®зҡ„жҺҘеҸЈпјҢеҸҜд»ҘзңӢеҲ°е…ҲжҳҜеҲӣе»әдәҶDefaultClientConfigImplй»ҳи®Өе®һзҺ°зұ»пјҢеҶҚconfig.loadProperties(this.name)еҠ иҪҪеҪ“еүҚClientзӣёе…ізҡ„й…ҚзҪ®

ж №жҚ®еҰӮдёҠжіЁйҮҠпјҢеҰӮжһңдҪ жІЎжңүеңЁйЎ№зӣ®дёӯжҢҮе®ҡribbonзӣёе…ій…ҚзҪ®пјҢйӮЈд№ҲдјҡдҪҝз”ЁDefaultClientConfigImplдёӯзҡ„й»ҳи®ӨйқҷжҖҒй…ҚзҪ®пјҢеҰӮжһңSpring enviromentдёӯеҢ…еҗ«вҖңribbon.й…ҚзҪ®йЎ№вҖқиҝҷзұ»й’ҲеҜ№жүҖжңүClientзҡ„й…ҚзҪ®дјҡиў«еҠ иҪҪиҝӣжқҘпјҢжңүвҖңclientеҗҚ.ribbon.й…ҚзҪ®йЎ№вҖқиҝҷзұ»й’ҲеҜ№жҹҗдёӘClientзҡ„й…ҚзҪ®дҝЎжҒҜд№ҹдјҡиў«еҠ иҪҪиҝӣжқҘ

йқҷжҖҒй…ҚзҪ®еҰӮдёӢпјҡ

RibbonClientж ёеҝғжҺҘеҸЈе®һзҺ°зұ»й…ҚзҪ®еҠ иҪҪеҸҠеҲӣе»ә

дёҠйқўиҜҙе®ҢIClientCOnfigй…ҚзҪ®йЎ№жҳҜеҰӮдҪ•еҠ иҪҪзҡ„пјҢжҢүйҒ“зҗҶиҜҙе…¶дёӯе·Із»ҸеҢ…еҗ«дәҶеҪ“еүҚRibbonClientдҪҝз”Ёе“ӘдёӘж ёеҝғжҺҘеҸЈе®һзҺ°зұ»зҡ„й…ҚзҪ®пјҢдҪҶSpring CloudеңЁжӯӨеӨ„е®ҡд№үдәҶиҮӘе·ұзҡ„е®һзҺ°йҖ»иҫ‘

дёӢйқўзңӢзңӢPropertiesFactoryзҡ„йҖ»иҫ‘

ж•…д»ҘдёҠйқўеҲӣе»әIRuleжҺҘеҸЈе®һзҺ°зұ»зҡ„йҖ»иҫ‘

  • е…ҲйҖҡиҝҮpropertiesFactoryжҹҘзңӢSpring enviromentдёӯжҳҜеҗҰй…ҚзҪ®дәҶй’ҲеҜ№еҪ“еүҚRibbon Clientзҡ„IRuleж ёеҝғжҺҘеҸЈе®һзҺ°зұ»зҡ„й…ҚзҪ®дҝЎжҒҜпјҢеҰӮжңүпјҢе°ұеҲӣе»әе…¶е®һдҫӢиҝ”еӣһпјҲзӣёе…ій…ҚзҪ®ж јејҸпјҡ clientName.ribbon.NFLoadBalancerRuleClassName=е…·дҪ“IRuleе®һзҺ°зұ»пјү
  • еҰӮжІЎжңүпјҢйӮЈд№ҲжІЎжңүзӣҙжҺҘдҪҝз”ЁNetflixеңЁе…¶DefaultClientConfigImplдёӯзҡ„йқҷжҖҒй…ҚзҪ®пјҢиҖҢжҳҜдҪҝз”ЁSpring CloudиҮӘе®ҡд№үзҡ„й»ҳи®Өе®һзҺ°зұ»пјҢжӢҝIRuleи§„еҲҷжҺҘеҸЈжқҘиҜҙжҳҜZoneAvoidanceRule

жҖ»з»“пјҡ

йҰ–е…ҲдјҡеҲӣе»әRibbonClientзҡ„ApplicationContextдёҠдёӢж–ҮпјҢе№¶зЎ®е®ҡдҪҝз”Ёе“ӘдёӘConfigurationй…ҚзҪ®зұ»

1гҖҒ@RibbonClientsжіЁеҶҢзҡ„е…ЁеұҖй»ҳи®Өй…ҚзҪ®зұ»

2гҖҒ@RibbonClientжіЁеҶҢзҡ„жҹҗдёӘClientй…ҚзҪ®зұ»

3гҖҒSpring Cloud й»ҳи®Өзҡ„RibbonClientConfigurationй…ҚзҪ®зұ»

зЎ®е®ҡй…ҚзҪ®зұ»еҗҺе°ұжҳҜеҠ иҪҪClientзӣёе…ізҡ„IClientConfigй…ҚзҪ®дҝЎжҒҜпјҢе№¶еҲӣе»әж ёеҝғжҺҘеҸЈе®һзҺ°зұ»

еҰӮжһңжІЎжңүиҮӘе®ҡд№үе…ЁеұҖ/е®ўжҲ·з«Ҝй…ҚзҪ®зұ»пјҢйӮЈд№Ҳе°ұжҳҜдҪҝз”ЁRibbonClientConfigurationпјҢиҖҢе…¶и§„еҲҷжҳҜ

еҜ№дәҺи¶…ж—¶зӯүй…ҚзҪ®пјҲйҷӨж ёеҝғжҺҘеҸЈе®һзҺ°зұ»д»ҘеӨ–пјүпјҡдҪҝз”ЁNetflixзҡ„й…ҚзҪ®йҖ»иҫ‘пјҢйҖҡиҝҮ ribbon.xxx дҪңдёәй»ҳи®Өй…ҚзҪ®пјҢд»Ҙ clientName.ribbon.xxx дҪңдёәе®ўжҲ·з«Ҝе®ҡеҲ¶й…ҚзҪ®

еҜ№дәҺж ёеҝғжҺҘеҸЈе®һзҺ°зұ»й…ҚзҪ®пјҡе®ўжҲ·з«Ҝе®ҡеҲ¶й…ҚзҪ®д»Қз„¶дҪҝз”Ё clientName.ribbon.xxxпјҢдҪҶй»ҳи®Өй…ҚзҪ®жҳҜSpring CloudеңЁRibbonClientConfigurationж–№жі•дёӯеҶҷжӯ»зҡ„й»ҳи®Өе®һзҺ°зұ»

е·Із»ҸзҹҘйҒ“еӨ§жҰӮзҡ„йҖ»иҫ‘дәҶпјҢдёӢйқўе°ұзңӢзңӢе…·дҪ“еҰӮдҪ•иҮӘе®ҡд№үClientй…ҚзҪ®гҖҒе…ЁеұҖй…ҚзҪ®

еҰӮдҪ•иҮӘе®ҡд№үRibbonClientй…ҚзҪ®гҖҒе…ЁеұҖй…ҚзҪ®

иҝҷйғЁеҲҶеңЁSpring Cloudе®ҳж–№referenceдёӯжңүиҜҙжҳҺ 16.2 Customizing the Ribbon Client

еӨ§иҮҙж„ҸжҖқеҰӮдёӢпјҡ

  • дёҖйғЁеҲҶй…ҚзҪ®пјҲйқһж ёеҝғжҺҘеҸЈе®һзҺ°зұ»зҡ„й…ҚзҪ®пјүеҸҜд»ҘдҪҝз”ЁNetflixеҺҹз”ҹAPIжҸҗдҫӣзҡ„ж–№ејҸпјҢеҚідҪҝз”ЁеҰӮ *.ribbon.** зҡ„ж–№ејҸй…ҚзҪ®пјҢе…·дҪ“жңүе“Әдәӣй…ҚзҪ®йЎ№пјҢеҸҜд»ҘеҸӮиҖғ com.netflix.client.config.CommonClientConfigKey

  • еҰӮжһңжғіжҜ”иҫғе…Ёйқўзҡ„жҺ§еҲ¶RibbonClientе№¶ж·»еҠ дёҖдәӣйўқеӨ–й…ҚзҪ®пјҢеҸҜд»ҘдҪҝз”Ё @RibbonClient жҲ– @RibbonClients жіЁи§ЈпјҢе№¶й…ҚзҪ®дёҖдёӘй…ҚзҪ®зұ»пјҢеҰӮдёҠзҡ„ FooConfiguration

    • @RibbonClient(name = вҖңfooвҖқ, configuration = FooConfiguration.class) жҳҜй’ҲеҜ№еҗҚдёә foo зҡ„RibbonClientзҡ„й…ҚзҪ®зұ»пјҢд№ҹеҸҜд»ҘдҪҝз”Ё@RibbonClients({@RibbonClientж•°з»„}) зҡ„еҪўејҸз»ҷжҹҗеҮ дёӘRibbonClientи®ҫзҪ®й…ҚзҪ®зұ»

    • @RibbonClients( defaultConfiguration = { xxx.class } ) жҳҜй’ҲеҜ№жүҖжңүRIbbonClientзҡ„й»ҳи®Өй…ҚзҪ®

      • е®ҳж–№ж–ҮжЎЈиҜҙ FooConfigurationй…ҚзҪ®зұ» еҝ…йЎ»жҳҜ@Configurationзҡ„пјҢиҝҷж ·е°ұеҝ…йЎ»жіЁж„ҸпјҢSpringBootдё»еҗҜеҠЁзұ»дёҚиғҪжү«жҸҸеҲ°FooConfigurationпјҢеҗҰеҲҷй’ҲеҜ№жҹҗдёӘRibbonClientзҡ„й…ҚзҪ®е°ұдјҡеҸҳжҲҗе…ЁеұҖзҡ„пјҢеҺҹеӣ жҳҜеңЁеҲӣе»әжҜҸдёӘRibbonClientж—¶дјҡдёәе…¶еҲӣе»әApplicationContextдёҠдёӢж–ҮпјҢе…¶parentе°ұжҳҜдё»еҗҜеҠЁзұ»еҲӣе»әзҡ„ApplicationContextпјҢеӯҗApplicationContextдёӯеҸҜд»ҘдҪҝз”ЁзҲ¶ApplicationContextдёӯзҡ„BeanпјҢдё”еҲӣе»әBeanж—¶йғҪдҪҝз”ЁдәҶ@ConditionalOnMissingBeanпјҢжүҖд»ҘFooConfigurationеҰӮжһңиў«дё»еҗҜеҠЁзұ»зҡ„дёҠдёӢж–ҮеҠ иҪҪпјҢдё”еҲӣе»әдәҶжҜ”еҰӮIRuleзҡ„е®һзҺ°зұ»пјҢеңЁжҹҗдёӘRIbbonClientеҲӣе»әе…¶еӯҗApplicationContextе№¶@BeanжғіеҲӣе»әе…¶иҮӘе®ҡд№үIRuleе®һзҺ°зұ»ж—¶пјҢдјҡеҸ‘зҺ°parent ApplicationContextе·Із»ҸеӯҳеңЁпјҢе°ұдёҚдјҡеҲӣе»әдәҶпјҢй…ҚзҪ®е°ұеӨұж•ҲдәҶ

        дҪҶеңЁжҲ‘зҡ„е®һйӘҢдёӯпјҢеҚідҪҝFooConfigurationдёҚеҠ @ConfigurationжіЁи§Јд№ҹеҸҜд»ҘеҠ иҪҪдёәRibbonClientзҡ„й…ҚзҪ®пјҢдё”з”ұдәҺжІЎжңү@ConfigurationдәҶпјҢд№ҹдёҚдјҡиў«дё»еҗҜеҠЁзұ»жү«жҸҸеҲ°

жүҖд»Ҙдё»иҰҒеҲҶжҲҗ2з§Қй…ҚзҪ®пјҡ

пјҲ1пјүи¶…ж—¶ж—¶й—ҙзӯүйқҷжҖҒй…ҚзҪ®пјҢдҪҝз”Ё ribbon.*** й…ҚзҪ®жүҖжңүClientпјҢдҪҝз”Ё *.ribbon. й…ҚзҪ®жҹҗдёӘClient

пјҲ2пјүдҪҝз”Ёе“Әз§Қж ёеҝғжҺҘеҸЈе®һзҺ°зұ»й…ҚзҪ®пјҢдҪҝз”Ё**@RibbonClientsжіЁи§ЈеҒҡй»ҳи®Өй…ҚзҪ®пјҢдҪҝз”Ё@RibbonClient**еҒҡй’ҲеҜ№Clientзҡ„й…ҚзҪ®пјҲжіЁж„Ҹ@ConfigurationдёҚиҰҒиў«SpringBootдё»еҗҜеҠЁзұ»жү«жҸҸеҲ°зҡ„й—®йўҳпјү
